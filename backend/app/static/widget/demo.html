Widget demo page

This page loads the embeddable widget for a real bot. Open it at http://localhost:8008/widget/demo.html — the chat bubble appears in the bottom-right corner.

The bubble shows because this bot's Website URL is blank (no domain lock) and its display rule is “all pages”. To actually get replies, enable a provider with a valid model + API key for this bot in the panel (Providers tab); otherwise sending a message returns a “not configured yet” notice.
